# H5 前台 + 管理后台

## 本地启动

双击 `start.bat`，或在当前目录运行：

```bash
node server.js
```

打开：

- 前台：`http://localhost:3000/`
- 后台：`http://localhost:3000/admin.html`

默认后台密码：`admin123`

## 云服务器启动

服务器需要安装 Node.js 18 或以上版本。

```bash
cd 项目目录
ADMIN_PASSWORD=你的强密码 PORT=3000 node server.js
```

后台资料保存在：

```text
data/site-data.json
```

每次后台保存时，会自动在同目录生成一份 `.bak` 备份文件。

## 后台能管理的内容

- 基础信息
- 学历规划师，支持继续新增，前台横向滑动展示
- 新媒体账号，支持继续新增，前台横向滑动展示
- 项目类目
- 学校资料和学校图片 URL
- 专业、专科/本科价格、专业详情

## 图片替换

后台已支持直接上传图片：

- 点击上传区域选择图片
- 将图片拖入上传区域
- 复制图片后，在后台页面粘贴上传
- 单张图片最大 4MB
- 支持 JPG、PNG、WebP、GIF、SVG

上传后后台会显示预览，确认无误后点击“保存全部”。

当前可替换的图片项：

- 首页 Banner 图片
- 页面背景图片
- 学历规划师头像
- 新媒体账号头像
- 项目封面图片
- 学校头像 / 学校介绍图

上传后的图片保存在：

```text
uploads/
```

上云备份时，除了 `data/site-data.json`，也要一起备份 `uploads/` 文件夹。
